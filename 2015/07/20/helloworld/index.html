Algae-html5项目代码说明

代码逻辑

使用到的开源项目

  • angular-seed

    AngularJS官方推荐的种子项目,做了适当的删减(例如删去了版本信息、测试代码等)

  • angular-bootstrap

    angular-ui出品的bootstrap的angular编译后版本，可以在不适用jQuery工具库的前提上，使用angular的方式来使用bootstrap组件和功能

  • angular-ui-router

    增强angular的router功能

  • angular-loading-bar

    页面顶端数据加载进度条，在异步加载数据时自动显示进度条

  • D3

    图表绘制，该项目中主要用来画折线图。

所有要用到的依赖包都已经下载源码及编译压缩后的代码都放在了bower_components文件夹下，目前项目中的引用全部指向本地文件，无外网时也可以使用

依赖包设置

项目中使用bower管理依赖包，见bower.json

全局参数

全局参数配置目前放在了app.js文件下，利用angular的contant引入。形如：

myapp.constant('urlConfig', {
    ReadPortsUrl:"http://algae.azurewebsites.net/api/ReadPorts"
});

在这里目前有如下配置：

接口URL地址配置

  • urlConfig.ReadPortsUrl
  • urlConfig.WritePortsUrl
  • urlConfig.JournalsUrl
  • urlConfig.SystemDetailsUrl
  • urlConfig.PortHistoriesUrl
  • urlConfig.ControlConfigUrl

参数配置

  • params.loopTime:定时调用间隔时间
  • params.downSampling:按时间段取数据是否限定100个数据
  • params.maxNumberOfValue:实时监控保存数据的最多数量

多个监控器实现说明

在第一次取数据时，按照传感器portid创建相应的数组：

$scope.selectedArray[port_id] = {
    portid: data.PortId,
    isSelected: false,
    start_DateTime: new Date(),
    start_HourTime: new Date(),
    end_DateTime: new Date(),
    end_HourTime: new Date(),
    realtime: true,
    startopened: false,
    endopened: false
};

点击传感器以后就将对应数组的isSelected设为true,然后前台渲染

其他参数说明：

  • realtime:是否实时监控
  • start_DateTime,end_DateTime:时间选择范围
  • startopened,endopened:日期选择框是否打开

实时监控图表说明

每次获取新数据以后就推送至对应的图表实例当中

//遍历数组向图表写入数据
for (var selected in $scope.selectedArray) {
    if ($scope.selectedArray[selected] != undefined) {  //检查是否存在该port
        if ($scope.selectedArray[selected].isSelected == true) {  //过滤是否选中
            if ($scope.selectedArray[selected].realtime == true) {  //过滤是否实时显示
                updateChart($scope.selectedArray[selected].portid);
            }
        }
    }
}

功能说明

MONITOR页面

监控Sensors和Controls的当前状态

  • 实时显示传感器状态

按照固定时间间隔调用查询接口，将传感器当前的状态实时显示在页面上。如果isConnected为false，那么这个传感器会标红显示出来。

  • 多个传感器状态图表监控

可以通过点击传感器增加或者删除监控对象，监控对象会在页面右侧显示实时的监控图表。监控图标监控范围可以通过调整params.loopTime和params.maxNumberOfValue来实现。

  • 切换是否实时

图表右上角有一个‘realtime’的checkbox，默认选中。选中时就会实时监控，不选中就不会插入新数据。再次选中会清空当前图表重新开始监控。

  • 查询一定时间段内的数据

在图表上方可以选择时间段，然后点击‘query’按钮进行查询。查询以后会自动停止实时监控。

CONTROL页面

显示并操作切换Controls的模式（无接口，暂未实现）

JOURNAL页面

备忘录页面

  • 增加备忘录

点击左上角‘+’号按钮可以增加备忘录

  • 删除备忘录

点击对应备忘录右上角的‘x’按钮可以删除备忘录

  • 修改备忘录

点击正文部分，修改内容时会显示‘Save’按钮，点击‘Save’按钮可以保存当前内容

CALIBRATE页面

显示Controls的各项数据，同时可以设置当前值为最大值或者最小值

SYSTEM页面

显示system的信息，成功获取到系统信息以后会提示“Hardware is Connected”
